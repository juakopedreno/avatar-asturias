# Checklist de visibilidad PRTR / NextGenerationEU

## Objetivo

Verificar que la solucion cumple con la obligacion de comunicacion y visibilidad del PSTD/PRTR.

## UI publica y admin

- [ ] Mencion textual obligatoria visible:  
      `Actuacion financiada por la Union Europea – NextGenerationEU en el marco del Plan de Recuperacion, Transformacion y Resiliencia.`
- [ ] Referencias visibles a UE NextGenerationEU, PRTR y PSTD Torremolinos.
- [ ] Enlace visible a informacion oficial del PRTR.
- [ ] Evidencia fotografica/capturas de web ciudadana y panel admin.

## Entregables documentales

- [ ] Informes tecnicos incluyen bloque PRTR de identidad visual y texto obligatorio.
- [ ] Memorias y manuales incluyen referencia expresa a financiacion.
- [ ] Export tecnico incluye fecha y trazabilidad del bundle.

## Trazabilidad

- [ ] Las capturas y evidencias se archivan con fecha y version de despliegue.
- [ ] Se registra responsable de validacion previa a cada entrega.
