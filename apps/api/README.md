# @avatar/api

Backend NestJS base para:

- autenticacion y MFA inicial,
- gestion de usuarios y roles,
- contenidos y fuentes,
- politica de entrenamiento,
- analitica y auditoria,
- endpoints de salud.

## Comandos

```bash
npm run start:dev -w @avatar/api
npm run build -w @avatar/api
npm run lint -w @avatar/api
npm run test -w @avatar/api
```
