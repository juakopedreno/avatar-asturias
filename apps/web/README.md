# Avatar Turistico Torremolinos

Frontend en React + Vite para:

- Experiencia de avatar conversacional para usuarios.
- Backoffice administrativo (contenidos, fuentes, entrenamiento, auditoria y analitica).

## Requisitos

- Node.js 20+
- npm 10+

## Comandos

```sh
npm install
npm run dev
```

Comandos utiles:

```sh
npm run build
npm run preview
npm run test
npm run lint
```

## Estructura principal

- `src/pages`: pantallas publicas y panel admin.
- `src/components`: layout y componentes UI.
- `src/data/mockData.ts`: datos mock temporales para prototipo.

## Estado actual

Este repositorio contiene un prototipo funcional de interfaz. El siguiente paso es conectar APIs reales, autenticacion, almacenamiento y pipeline de IA (RAG) para cumplir requisitos de licitacion.
