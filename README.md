# Avatar Torremolinos Monorepo

Plataforma para asistente virtual con IA y backoffice administrativo.

## Workspaces

- `apps/web`: frontend React (canal ciudadano + panel admin).
- `apps/api`: backend NestJS (en construccion).
- `packages/shared-types`: contratos compartidos.
- `infra`: base de IaC multicloud.
- `docs`: documentacion tecnica y evidencias.

## Comandos

```bash
npm install
npm run dev:web
npm run dev:api
npm run lint
npm run test
npm run build
```
